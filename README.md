# OTT Audience Intelligence Dashboard

## Project Overview

The OTT Audience Intelligence Dashboard is a Data Analytics project focused on analyzing content trends across major OTT platforms including Netflix, Amazon Prime, Hulu, and Disney+.

The project uses Python, SQL, Pandas, NumPy, Matplotlib, and MySQL to perform data cleaning, exploratory data analysis (EDA), visualization, and business insight generation from a dataset containing 22,998 OTT titles.

---

## Objectives

* Analyze content distribution across OTT platforms
* Compare Movies and TV Shows
* Identify top-performing genres
* Analyze content ratings and audience categories
* Study content growth trends over time
* Discover country-wise content production patterns
* Perform SQL-based business analysis

## Key Insights

* Amazon Prime contains the largest content library with 9,668 titles.
* Netflix follows closely with 8,807 titles.
* Movies dominate the dataset compared to TV Shows.
* Drama is the most popular genre across platforms.
* The United States contributes the highest number of titles.
* Significant content growth is observed between 2015 and 2021.
* TV-MA is the most common content rating.
